# Установка и удаление

## Что входит в архив релиза

- папка `overlay apk` со всеми собранными overlay APK
- `adb` и сопутствующие файлы
- `install_win.bat`
- `install_mac.sh`
- `uninstall_win.bat`
- `uninstall_mac.sh`
- `disable-verity_win.bat`
- `disable-verity_mac.sh`
- краткая инструкция

## Перед установкой

- Убедитесь, что у вас есть доступ к автомобилю по `adb`.
- Убедитесь, что используете архив из стабильного релиза.
- Для обычной установки достаточно `install_win.bat` или `install_mac.sh`: скрипт сам выполнит подготовку verity.

## Установка на Windows

1. Распакуйте архив релиза.
2. Подключите автомобиль.
3. Запустите `install_win.bat`.
4. Следуйте подсказкам скрипта во время перезагрузок и повторного подключения.
5. Дождитесь завершения копирования, перезагрузки и активации overlays.

## Установка на macOS

1. Распакуйте архив релиза.
2. Подключите автомобиль.
3. При необходимости выполните `chmod +x adb install_mac.sh uninstall_mac.sh`.
4. Запустите `install_mac.sh`.
5. Следуйте подсказкам скрипта во время перезагрузок и повторного подключения.

## Удаление

Для удаления используйте:

- `uninstall_win.bat` на Windows
- `uninstall_mac.sh` на macOS

Скрипт отключит overlays, удалит overlay APK и перезагрузит устройство.

## Важно

- Не смешивайте файлы из разных релизов.
- Не копируйте overlay APK вручную, если не понимаете, что именно делаете.
- Для обычного использования берите только стабильные релизы из основного репозитория.
